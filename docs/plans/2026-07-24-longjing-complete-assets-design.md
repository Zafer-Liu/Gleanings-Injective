# 第二章完整玩法素材设计

## 范围

本次补齐《拾遗》第二章“西湖龙井”四幕中的全部玩法素材，不制作两章片尾短片素材。现有四张地图与四枚彩色徽章保留；补齐角色、可交互物件、采茶状态、炒茶状态和章节完成页的正式呈现。

## 视觉规则

- 延续第一章纯 2D 俯视 RPG 像素画：32×32 瓦片、32×48 角色、硬边透明、整数倍最近邻缩放。
- 使用 `fujian_rpg_24` 的暖炭黑、旧木褐、米纸色、朱红、灰蓝与低饱和茶绿；不加环境光、渐变、模糊或伪 3D 阴影。
- 角色保持三列四行行走图规范：向下、向左、向右、向上；每行依次为左步、站立、右步。
- 交互物件轮廓必须能在场景暗色地面上被识别，任务箭头只负责指引，不代替物件本体。

## 素材组成

### 专属角色

- 现代陈守一：旧茶坊主人，灰发、茶褐外套、克制站姿。
- 青年陈守一：深蓝短褂、布围裙，和现代形象保持可辨认的脸型与配色关联。
- 何师傅：年长制茶师，灰蓝布衣、深色围裙。
- 旧货摊主：褐色夹衣、旧帽，轮廓与第一章人物区分。
- 收货人：灰褐长衫、布包，外形不做反派脸谱化。

### 调查与任务物件

- 市场：茶罐甲、茶罐乙、随货卡、市场旧账、旧茶勺。
- 茶园：竹茶篓。
- 工坊与真相幕：旧签样、拒签留底、原始批次账、来源声明板。
- 物件以独立透明 PNG 覆盖在地图交互坐标上；完成拾取后按剧情状态隐藏，防止“箭头指着空气”。

### 茶芽与炒茶状态

- 五类茶芽：合适嫩芽、过嫩芽、成熟叶、带露湿叶、破损叶。每类使用独立轮廓和局部颜色，不只依赖色相。
- 炒茶以锅内叶片视觉层呈现青锅、回潮、辉锅三阶段。状态随 `firingStep`、热度、含水和整形数据变化；叶色、卷曲、碎屑与蒸汽提示同步更新。
- 选择面板上方显示正式锅叶素材，避免只有文字选择而没有视觉反馈。

## 场景接入

- `BootScene` 统一预载第二章专属角色、物件、茶芽、炒茶状态和结算徽章，加载失败继续沿用现有错误屏。
- `LongjingMarketScene`、`LongjingTerraceScene`、`LongjingWorkshopScene`、`LongjingTruthScene` 使用专属人物，不再复用首章角色并染色。
- 新增轻量物件层，根据幕和阶段渲染任务物件；交互坐标仍以现有内容数据为唯一来源，避免视觉与碰撞分叉。
- 茶园把程序绘制三角形替换为五类纹理图片。
- 章节完成页用已有彩色徽章 PNG 替代矩形和“坛/芽/叶”文字占位。

## 验收

- 所有新增 PNG 通过尺寸、透明通道、硬透明和项目调色板检查。
- 第二章人物不再引用 `actor-afeng`、`actor-azhen` 或 `actor-taipo-young`。
- 当前任务箭头下必须有角色或物件；茶芽五态可从轮廓判断；炒茶三阶段有明显视觉变化。
- 完成页展示彩色徽章原图，并保持小尺寸清晰。
- 全量单元测试和生产构建通过；四幕可连续完成，不改变现有存档结构与碰撞逻辑。
