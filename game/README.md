# 《拾遗》可玩章节

当前版本包含两个连续可玩的完整章节。第一章《一坛回声》从太婆寄来的纸箱进入福建老酒
家族记忆，并以约 60 秒的黄酒文化后记收束；第二章《一叶来处》从杭州茶市的一组同号茶罐
出发，经过西湖茶园、手工炒制记忆与来源核验，最后衔接约 60 秒的西湖龙井文化后记。

## 启动

需要 Node.js 20+ 与 pnpm 10+。

```bash
cd game
pnpm install
pnpm dev
```

浏览器打开终端显示的地址。生产构建使用：

```bash
pnpm build
pnpm exec vite preview
```

## 操作

- `WASD` / 方向键：四方向移动
- `E` / 空格：调查、推进对白、确认
- `I`：打开或关闭背包
- 酒坛前长按 `E`：揭坛
- 文化后记 `Space`：暂停，`S`：开关字幕，`↑` / `↓`：音量，`M`：静音，`Esc`：跳过
- 第一章完成页 `N`：进入第二章
- 第二章完成页 `V`：重看龙井后记，`H`：重看黄酒后记，`R`：重新体验第二章

## 第一章 · 一坛回声

1. 《开坛》：林念安整理太婆纸箱、阅读家书并揭开老酒坛。
2. 《冬酿》：青年太婆在阿凤师的酒坊里辨红曲、拌曲、下缸与封坛。
3. 《添暖》：中年太婆备齐碗、面线和老酒，为阿珍煮一碗老酒面线。
4. 《归家》：林念安和米娅一起整理三代人的文化标签，并完成本地演示铸造。
5. 《一坛酒，一条河》：从家中酒坛讲到谷物酿酒、贾湖遗存、酒曲复式发酵、宋元工艺与
   福建红曲黄酒。片中明确区分贾湖发酵饮品与今天的黄酒。

## 第二章 · 一叶来处

1. 《同号茶罐》：林念安和米娅在雨后茶市调查两罐同号茶，收集批次、日期与流转记录，
   并在来源板上区分“声明、记录、冲突、待核”。
2. 《清明之前》：进入西湖茶园，逐片观察十二枚鲜叶；玩家可选择采或留，错误只给出
   观察反馈，不中断流程。
3. 《掌心与锅》：在何师傅指导下完成五轮手工炒制判断。玩法强调看叶、听声、感受阻力，
   不把十种传统手法误写成固定招式连段。
4. 《名字有来处》：回到封存的旧作坊补齐原始批次、旧签名和拒绝冒名记录，完成来源板，
   并选择“还名、留真、传手”之一作为结章题记。
5. 《一片叶，为什么叫西湖龙井》：从产区、清明采摘与手工炒制讲到十八棵御茶传说、
   传统制茶技艺及其相关习俗。片中明确区分传说与可核验史实。

当前任务物件上方始终只有一个金色箭头。第二、三幕地图均大于 640×360 视口，摄像机跟随
人物移动；角色使用固定 32×48 像素帧，不会因向右移动而缩小。

## 内容与代码位置

- `src/content/act1` 至 `src/content/act4`：中文对白、任务、交互点与地图几何
- `src/content/film/huangjiuFilm.ts`：文化后记时间轴、字幕与来源链接
- `src/content/longjing/`：第二章地图、对白、任务、十二枚鲜叶与炒制内容
- `src/content/film/longjingFilm.ts`：龙井文化后记时间轴、字幕与来源链接
- `src/game/domain/`：可测试的章节状态机与启动路由
- `src/game/scenes/`：两章探索幕、两支文化后记与章节完成场景
- `../assets/rpg_v2/`：运行时 24 色像素素材；`*_source.png` 只用于回溯

对白、任务和交互均由内容文件驱动。新增文案时保留稳定的 `id`，人物显示名只写中文。
改变碰撞或交互坐标后，需同时确认目标仍可从相邻 tile 接近。

## 存档

章节存档键为 `gleanings.chapter-one.save.v2`。完成第一幕后，旧键
`gleanings.act1.save.v1` 会自动迁移到第二幕；刷新后会按 `currentAct` 回到对应场景。
坏存档会备份到 `gleanings.chapter-one.corrupt`。

第二章存档键为 `gleanings.chapter-two.save.v1`，坏存档会备份到
`gleanings.chapter-two.corrupt`。启动时如果已有第二章进度，会优先恢复第二章。

手动重置可在浏览器控制台执行：

```js
localStorage.removeItem("gleanings.chapter-one.save.v2");
localStorage.removeItem("gleanings.act1.save.v1");
localStorage.removeItem("gleanings.chapter-two.save.v1");
location.reload();
```

## Injective 勋章存证演示

已拾取的剧情道具会立刻同步到右上角的「收藏馆」；第一幕完成后还会加入「冬酿守忆章」。收藏馆无需连接钱包即可使用。玩家可对任意收藏主动选择「上链展示」，确认交易后该收藏会作为 ERC-721 NFT 展示链上编号。

先在另一个终端启动链上桥服务，并完成 `rpg-chain-kit/.env` 中 Injective EVM RPC、Chain ID、合约地址和浏览器地址的真实网络配置：

```powershell
cd ..\..\rpg-chain-kit
npm start
```

可选环境变量 `VITE_CHAIN_BRIDGE_URL` 默认是 `http://127.0.0.1:3100`。演示前请在 MetaMask 中切换到已部署合约的 Injective EVM 网络。
